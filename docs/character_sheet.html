The three classes, walking (8×)

Squad color stays the whole torso — friend-or-foe at a glance outranks costume (the concept board keeps team identity as color marks too). Class is worn on top: steel helm + pauldrons + shield + sword; forest hood + quiver + bow; leather cap + tool pack + crossbow. The helmet leads the facing, boots step along the move direction (distance-driven — a standing body holds still), and the weapon doubles as the aim line.

Combat poses (fighter, 6×) + ranged reload read

The sword pulls back on windup (the dodgeable telegraph — drawn inside the existing red wedge), crosses on the strike, tucks behind the raised shield on block (the 120° arc stays the authoritative cover telegraph). For your own archer, the nocked arrow / loaded bolt vanishes with the shot and returns when the fire gate re-arms.

Friend or foe: 4 squads × 3 classes (4×)

The read that must never regress. Gear tones are deliberately too muted to fight a squad color; the ranger's hood green is darker and grayer than squad-green.

True scale (1× and 2×) — before vs after

In game a body is ~10px across (radius 0.4u × TILE 12). Left of the divider: the old placeholder (disc + facing tick). Right: the soldiers. Same hitbox, same disc — presence comes from shadow, helm, gear and gait, not size.

Armory palette

Pulled toward the concept board's tone strip: steel, oiled leather, bow wood, boot black. Squad colors are untouched.

What changed / what didn't

  • Render-only. Zero sim / protocol / config-hash changes — facing (ax/ay), class and state bits were already in every snapshot.
  • Kept intact: hp bars, name labels, damage flash, block arc, swing wedges, dash streak, banking beacon, root shackles, carry diamond (now sways with the gait), squad disc size = honest hitbox.
  • Removed: the abstract class marks (inner ring / square) and the black facing tick — helm + weapon carry both jobs now.
  • New knobs in fx/config.ts → FX.character: shadowAlpha, strideAmp, bootR.
  • New for your own archer: OwnVisual.atkReady mirrors the local fire gate → visible re-nock on reload.
